Class Interval

A module to safely handle intervals with a name

Usage:

import { Interval } from '@queelag/core'

Interval.start('ACTIVITY', () => {
  // do something
}, 5000)

console.log(Interval.isRunning('ACTIVITY'))
// logs true

setTimeout(() => Interval.stop('ACTIVITY'), 10000)
